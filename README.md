# Lynko-Reto
Lynko surge en el contexto del sistema educativo actual en el que los estudiantes pueden presentar desmotivación hacia el aprendizaje debido a las formas tardicionales de enseñar.

Si estos estudiantes generan esa desmotivación, debido a que encuentran el aprendizaje aburrido, es lógico que el aprendizaje se vería afectado, bajando el desempeño de los estudiantes.

Una vez que se presente esta disminución del desempeño de los estudiantes, el problema escalaría a los salones de clase, donde ya la atención se empezaría a perder.

Esta plataforma busca:

Reducir el aburrimiento en el estudio
Aumentar la motivación por el estudio
Hacer el aprendizaje más accesible
Enseñar de forma práctica, no solo teórica
